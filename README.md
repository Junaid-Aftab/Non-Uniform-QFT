# Non-Uniform Quantum Fourier Transform (NUQFT)

This repository contains the numerical computations performed for the **Non-Uniform Quantum Fourier Transform (NUQFT)** project.

The NUQFT generalizes the standard Quantum Fourier Transform by relaxing the assumption of uniformly spaced input or frequency domains. This work focuses on the numerical implementation and evaluation of the proposed NUQFT algorithm, including simulations, validation experiments, and performance analysis.
